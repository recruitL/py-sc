# Chapter 2: Data Interpolation

Interpolation estimates unknown values from known discrete data points.

This chapter starts with three common methods:

1. Polynomial interpolation
2. Piecewise linear interpolation
3. Natural cubic spline interpolation

## Files

* `polynomial_interpolation.py`: Lagrange polynomial example
* `piecewise_interpolation.py`: piecewise linear example
* `cubic_spline_interpolation.py`: natural cubic spline example
* `compare_interpolation.py`: compare the three methods on the same data

## Run

From the repository root:

```bash
python chapters/ch02_interpolation/compare_interpolation.py
```

The scripts are intentionally compact so that each method can be read as part
of the chapter notes.
